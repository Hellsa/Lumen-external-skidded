#  Skidding — UI Recreation Project
 
> A work-in-progress pixel-perfect recreation of the **Lumen** interface design, built in **WPF** with **C#**.
 
---
 
##  Overview
 
This project is a personal attempt at recreating the **Lumen** UI from scratch using **WPF (Windows Presentation Foundation)** and **C#**. The goal is to match the original as closely as possible in terms of layout, colors, sizing, and overall structure. While the project is still heavily in progress, it serves as a learning exercise in UI replication and desktop application development.

##  Known Issues You Gonna See
 
The recreation currently has several notable discrepancies when compared to the original. All of these are known and fixable:
 
###  Color Errors
- Several colors are off — background tones, accent colors, and text colors don't fully match the original palette.
- Some gradients and transparency values are either missing or approximated.
- The overall color temperature feels slightly different from the target.
###  Sizing & Spacing Issues
- Component sizes (buttons, inputs, cards) are not pixel-accurate yet.
- Padding and margin values deviate from the original in multiple sections.
- Font sizes and line-heights need fine-tuning to match the original typography.
###  Missing Sections
- Several UI sections present in the original have not been implemented yet.
- Some components are placeholders or stubs.
- Navigation elements and certain interactive states are incomplete.
